# ROS 2 Jazzy 学习项目

这是一个面向零基础学习者的 ROS 2 Jazzy 练习仓库。

当前目标不是一开始就做复杂项目，而是先把学习环境、常用命令、笔记结构整理好，再逐步创建 ROS 2 package 并练习节点通信。

## 目录说明

- `docs/`：放环境配置和常用命令说明
- `notes/`：放每天的学习笔记
- `src/`：以后存放 ROS 2 package

## 建议学习顺序

1. 理解 workspace、package、node 的基本概念
2. 学会进入 ROS 2 环境
3. 认识常用命令
4. 再开始创建第一个 Python package
